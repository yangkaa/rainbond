### 2.2 订阅过期状态页面（授权有效，但无法更新）

```
创建一个显示"订阅已过期"状态的插件中心页面：

【授权状态栏】
- 标签："高级版" (蓝色)
- 状态文字："订阅已过期" (橙色警告文字)
- 有效期信息："授权有效期至 2026-12-31"
- 右侧按钮："续费" (橙色/警告色) + "授权管理"

【状态栏下方警告横幅】
┌─────────────────────────────────────────────────────────┐
│  ⚠️ 您的订阅已过期，无法安装新插件或更新现有插件          │
│     已安装的插件可继续使用                [联系续费]      │
└─────────────────────────────────────────────────────────┘

【插件列表行为】
- 已安装插件：正常显示，可使用
- 未安装插件：
  - "安装" 按钮禁用并置灰
  - 悬停提示："订阅已过期，无法安装"
- 有可用更新的插件：
  - "升级" 按钮禁用
  - 只显示当前版本，隐藏升级箭头
  - 悬停提示："订阅已过期，无法升级"

【样式】
- 警告横幅：浅橙色背景 (#fff7e6)
- 禁用按钮：灰色 (#d9d9d9)
```

### 2.4 升级确认弹窗


### 2.5 卸载确认弹窗

```
创建一个带警告强调的卸载确认弹窗：

【弹窗标题】
- 标题："确认卸载「企业基础功能插件」？"
- 右侧关闭按钮 (X)

【警告区域】
突出的警告框，使用红色/橙色样式：
- 背景：浅红色 (#fff2f0)
- 边框：红色 (#ffccc7)
- 图标：⚠️ 警告三角

警告信息（每条单独一行）：
- "卸载后相关功能将不可用"
- "插件数据将被清除"

【操作按钮】
- "取消" 按钮 (轮廓样式，左侧)
- "确认卸载" 按钮 (红色/危险色 #ff4d4f，右侧)

【注意】确认按钮应为红色，表示危险操作，不是蓝色

【弹窗尺寸】约 400px 宽度
```

### 2.6 授权验证失败提示

```
为授权管理弹窗的"更新授权"区域创建 3 种错误状态变体：

【变体 1：格式错误】
- 输入框边框变红色 (#ff4d4f)
- 输入框下方显示错误信息，红色文字：
  - 图标：❌ 或 ⚠️
  - 文字："授权码格式不正确，请检查后重试"
- "验证并激活" 按钮保持可用以便重试

【变体 2：集群不匹配】
- 输入框边框变红色
- 错误信息："该授权码与当前集群不匹配"
- 辅助文字（灰色）："请确认授权码是否为当前集群申请"

【变体 3：授权已过期】
- 输入框边框变橙色 (#faad14)
- 警告信息："该授权码已过期，请联系销售续费"
- 显示过期时间："过期时间: 2025-01-01"

【通用样式】
- 错误文字颜色：#ff4d4f (红色)
- 警告文字颜色：#faad14 (橙色)
- 可添加轻微抖动动画表示错误
- 保持输入框内容可见，方便用户检查
```

### 2.7 权限不足提示弹窗

```
创建一个当用户尝试安装需要更高版本授权的插件时显示的弹窗：

【弹窗顶部】
- 图标：🔒 锁图标（大号，居中）
- 标题："权限不足"

【内容区域】
信息卡片，显示对比：
┌─────────────────────────────────────────┐
│  该插件需要「高级版」授权                │
│                                         │
│  当前授权: 基础版    ──────────          │
│  所需授权: 高级版    ✨                  │
└─────────────────────────────────────────┘

"高级版" 应该突出显示（蓝色或标签样式）

【说明文字】
"升级到高级版可解锁全部企业插件"

【操作按钮】
- "取消" 按钮 (轮廓样式，左侧)
- "联系销售升级" 按钮 (主色蓝色，右侧)

【可选】按钮下方添加 "查看版本对比" 链接

【弹窗尺寸】约 420px 宽度
【样式】内容居中，布局简洁
```

## 四、使用说明

1. 将提示词复制到 Lovable
2. 生成后可以要求调整：
   - 修改配色为品牌色
   - 添加响应式设计
   - 增加交互动画
3. 导出设计稿后与团队评审
